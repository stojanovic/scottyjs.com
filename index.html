Deploy static websites and single page apps to AWS S3 and CloudFront with a single command

✤ ✤ ✤
Source code Tutorial
✤ ✤ ✤

Install Scotty as a global package by running:

npm install scottyjs --global

Then deploy your current folder to AWS S3 and CloudFront as a website by running:

scotty --website

Or, with NPM v5 run Scotty without installing it globally with new NPX command:

npx scotty -- --website

If you don’t know npx or are confused by the 4 dashes, please read the npx docs.
